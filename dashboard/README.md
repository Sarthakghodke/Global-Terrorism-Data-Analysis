An interactive Power BI dashboard was created to analyze global terrorism trends
from 1970 to 2021 using the Global Terrorism Database (GTD).

### Dashboard Features
- Year-wise trend of terrorist incidents
- Region-wise and country-wise attack distribution
- Attack types and weapon categories
- Casualties analysis (killed & wounded)

### Note on Power BI File
The original Power BI (.pbix) file is not uploaded to this repository due to
GitHub file size limitations.

Instead, dashboard screenshots are provided in the `images/` folder to
demonstrate the insights and visualizations.

### Tools Used
- Power BI
- Python (for data preprocessing)
- Global Terrorism Database (GTD)
